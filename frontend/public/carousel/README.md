# Carousel Images

## Folder Structure

```
carousel/
├── desktop/
│   ├── carousel-desktop-1.jpg (or .png)
│   ├── carousel-desktop-2.jpg (or .png)
│   ├── carousel-desktop-3.jpg (or .png)
│   ├── carousel-desktop-4.jpg (or .png)
│   ├── carousel-desktop-5.jpg (or .png)
│   └── carousel-desktop-6.jpg (or .png)
└── mobile/
    ├── carousel-mobile-1.jpg (or .png)
    ├── carousel-mobile-2.jpg (or .png)
    ├── carousel-mobile-3.jpg (or .png)
    ├── carousel-mobile-4.jpg (or .png)
    ├── carousel-mobile-5.jpg (or .png)
    └── carousel-mobile-6.jpg (or .png)
```

## Instructions

1. Add 6 images for desktop view in the `desktop/` folder
2. Add 6 images for mobile view in the `mobile/` folder
3. Name them exactly as shown above
4. Supported formats: JPG, PNG, WebP
5. Recommended sizes:
   - Desktop: 1920x700px or 1920x800px
   - Mobile: 768x500px or similar aspect ratio

## Notes

- Images will automatically display once added with the correct names
- If images are not found, a gradient fallback will be shown
- The carousel auto-advances every 5 seconds
- Users can navigate using Previous/Next buttons or indicator dots
